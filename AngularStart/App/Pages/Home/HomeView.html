Info

This project can be used as a starting point for setting up a modern SPA-application using TypeScript and AngularJs.

Essentials

AngularJs
AngularJs is a complete Javascript SPA framework. All you need and all you don't know you need within a SPA framework.
TypeScript
A superset to Ecmascript 5 incorperating classes and more from EcmaScript 6.. and of course Typings. The upcoming Angular 2.0 is actually written in TypeScript. Read more

Goodies

Less
Less is a css pre-compiler that eliminates much tedious css-work. Variables and functions (mixins) are some of the many advantages.
Bootstrap
Bootstrap is a css-framework built with less. Responsive with mobile-first makes your css-life soo easy. You can change the apperance with 'less' effort
Ui Bootstrap
Bootstrap components rewritten in pure AngularJS by the AngularUI Team. Bootstrap itself uses javascript for some richer components. This rewrite makes bootstrap component fit perfectly together with Angular JS. The original bootstrap-javascript is not needed.
FontAwesome
Nice solid icons to style your web. Built for bootstrap and styles written in less. Simple and awesome!
